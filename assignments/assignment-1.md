```c
    for(j=1; j<=n; j++)

```


## ۶. پیچیدگی و مرتبه زمانی قطعه کدهای زیر را بدست آورید

## الف
```c
for (i=1; i<=n; i++)
    for(j=i; j<=n-i; j++)
        w++;
```


## ب

```c
for (i=1; i<=n; i*=3)
    for(j=n; j>3; j--)
        w++;
```



## پ

```c
int n = 42;
for (i=1; i<=n; i++)
    for(j=i; j<=i; j++)
        w++;
```


## ج

```c
for (i=1; i<=n-1; i++)
    for(j=1; j<=n; j++)
        for(k=n; k>1; k/=2)
            w++;
```

```c
int Result = 0;
for (int i = 0; i < N; i++)
```
